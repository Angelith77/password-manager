# password-manager
Simple Python password manager (v1 &amp; v2)
A secure and simple command-line password manager built with Python.

---

## 🚀 Project Overview

This project allows users to safely store and manage their account credentials using a terminal-based interface.

It evolved across two versions:

---

## 📌 Versions

### 🔹 Version 1 - Basic System
- Single user system with master password
- Add, show, delete, and edit accounts
- Data stored locally in text files
- Simple authentication system

---

### 🔹 Version 2 - Advanced System
- Multi-user system with login & registration
- Each user has a separate file
- Passwords stored using SHA-256 hashing
- Add, show, delete, search, and edit accounts
- Improved security and structure

---
